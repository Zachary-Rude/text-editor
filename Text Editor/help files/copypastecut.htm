Cutting Text

To cut text, first select some text in the main text box, and then either select Edit > Cut or press Ctrl + X.

Copying Text

To copy text, first select some text in the main text box, and then either select Edit > Copy or press Ctrl + C.

Pasting Text

To paste text, either select Edit > Paste or press Ctrl + V after copying or cutting text.

Selecting All Text

To select all text, either select Edit > Select All or press Ctrl + A.
