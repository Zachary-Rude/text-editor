Saving Your Work

To save your work, either select File > Save* to save the currently edited file without creating a new file with the same contents**, or File > Save As...* to save your new text as a new file (or to save the current file under a new name).

* Note: You can also use their respective keyboard shortcuts, which are shown on the menu items.

** Note: If you try to use File > Save to save a new file, it will perform the same function as Save As.
