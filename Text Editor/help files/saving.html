Saving Your Work

In Notepad.NET, an asterisk character (*) appears before the file name in the title bar when the currently open file has unsaved changes. To save your work, either select File > Save or Ctrl + S to save the currently open file without creating a new file†, or File > Save As... or Ctrl + Shift + S to save the currently open file under a new name.

†Note: If you try to use File > Save or Ctrl + S to save a new file, it will perform the same function as File > Save As or Ctrl + Shift + S.
